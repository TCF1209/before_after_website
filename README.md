# DuoCode Before/After Website Demo

A stunning before/after website demo for **DuoCode Technology** — a single HTML file that toggles between a deliberately outdated "before" look and a cinematic, Azure Dragon-themed "after" experience.

## Quick Start

Open `index.html` in a browser. Click the toggle in the top-right corner to switch between Before and After modes.

## Features

- **Before Mode**: Grayscale, flat, boring — intentionally ugly
- **After Mode**: Cinematic Azure Dragon (青龙) canvas animation, glassmorphism cards, gradient text, smooth transitions
- **Toggle Transition**: Gold/white radial flash effect
- **Single File**: Pure HTML/CSS/JavaScript, no frameworks
